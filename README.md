## LeNet

- 利用 **FashionMNIST** 数据集（内置数据集），如需手动下载请访问 [FashionMNIST](https://pan.baidu.com/s/1dXc8cpsefWoUGaI3_bTRVQ?pwd=6666)。
- `python train.py` 训练模型，运行后会产生 `data`文件夹，放置数据集
- `python test.py`  测试模型
